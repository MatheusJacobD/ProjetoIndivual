# Jacob's Chess
Repositório criado para armazenar o projeto individual realizado nas aulas de Projeto e Inovação da SPTECH. O objetivo do projeto é medir o conhecimento obtido durante as aulas e bootcamps proporcionados.

## 🚀 O que o projeto contempla
- **HTML | CSS | JS**: O projeto possuí uma página estruturada que informa, testa e engaja usuários sobre xadrez.
- **Login | Cadastro**: Utilizando uma API fornecida pela faculdade, o projeto possuí uma tela de login e cadastro com conexão a um banco de dados hospedado em uma VM lubunto.
- **DashBoard**: Baseado nos resultados dos testes realizados por outros alunos, o projeto exibe os resultados no formato de tabelas e indices afim de informar o usuário.
